# VibeCheck Project
**Team:** [Elmo] & [Daryl]

## Overview
A Full-Stack web application that fetches fortunes, jokes, and tracks "smashes" using a Node.js backend.

## How to Run
1. Open terminal in `/backend` and run `npm install`.
2. Start server: `node index.js`.
3. Open `/frontend/index.html` in a browser.
